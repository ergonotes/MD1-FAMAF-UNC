**Definición 3.2.1.** Dados dos enteros $x$ e $y$ decimos que $y$ es un divisor de $x$,
y escribimos $y|x$, si para algún $q\in Z$ $$x=y\cdot q$$También decimos que $y$ es un factor de $x$, que $y$ divide a $x$, que $x$ es divisible
por $y$, y que $x$ es múltiplo de $y$.

**Proposición 3.2.2.** Sean $a, b, c$ enteros, entonces $$\begin{align*}
\text{a)} \quad & 1|a, \quad a|0, \quad a|\pm a; \\
\text{b)} \quad & \text{si } a|b, \text{ entonces } a|bc \text{ para cualquier } c; \\
\text{c)} \quad & \text{si } a|b \text{ y } a|c, \text{ entonces } a|(b + c); \\
\text{d)} \quad & \text{si } a|b \text{ y } a|c, \text{ entonces } a|(rb + sc) \text{ para cualesquiera } r, s \in \mathbb{Z}; \\
\text{e)} \quad & \text{si } a, b > 0 \text{ y } a|b, \text{ entonces } a \leq b.
\end{align*}$$Algunas **demostraciones**: 
**c**. como $a|b$, existe $q$ tal que $b = aq$. Análogamente, como $a|c$, existe $q′$ tal
que $c = aq′$. Entonces $b + c = aq + aq′$ $= a(q + q′)$, luego $a|(b + c)$.

**e**. como $a|b$ existe $q ∈ Z$ tal que $b = aq$, como $a, b >0$, entonces, por la compatibilidad de $>$ con el producto, $q > 0$. Luego $q − 1 ⩾ 0$ y, de nuevo por la compatibilidad, $a(q − 1) ⩾ 0$. Ahora bien, $b = aq = a + a(q − 1)$ luego $b − a = a(q − 1) ⩾ 0$, por lo tanto $b ⩾ a$.
