###### **Lema 4.3.1** Sea $p$ un número primo, entonces $$\begin{align*}
&\text{a) } p | \binom{p}{r}, \ \text{con } 0 < r < p, \\
&\text{b) } (a + b)^p \equiv a^p + b^p \pmod{p}.
\end{align*}$$**Demostración.**  $$\begin{align*}
&\text{a) Escribamos el número binomial de otra forma:} \\
&\binom{p}{r} = \frac{p!}{r! (p - r)!} = \frac{p \cdot (p - 1)!}{r! (p - r)!}.
\end{align*}$$es un número entero, digamos $k$, luego $$k \cdot r! (p - r)! = p \cdot (p - 1)!$$Como $p-1,r$ y $p-r$ son menores que $p$, entonces $(p−1)!, r!$ y $(p−r)!$ son producto de números menores que $p$ y por lo tanto son producto de primos menores que $p$. Por lo tanto, el primo $p$ no aparece en la descomposición prima de $(p−1)!, r!$ y $(p−r)!$, es decir, $p$ no los divide. Y por la igualdad, o por descarte, $p$ debe ser factor de $k = \binom{p}{r}$, luego $p|\binom{p}{r}.$

Para $\text{b)}$, sabemos por el [[2.5 Teorema del Binomio]] que $$(a + b)^p = \sum_{i=0}^{p} \binom{p}{i} a^i b^{p-i}$$$$\begin{align*}
&\text{Por a) es claro que } \binom{p}{i} a^i b^{p-i} \equiv 0 \pmod{p}, \ \text{si } 0 < i < p. \ \text{Luego,} \\
&(a + b)^p \equiv \binom{p}{0} b^p + \binom{p}{1} a b^{p-1} + \cdots + \binom{p}{p-1} a^{p-1} b^1 + \binom{p}{p} a^p \pmod{p} \\
&\equiv b^p + 0 + \cdots + 0 + a^p \pmod{p} \\
&\equiv b^p + a^p \pmod{p}
\end{align*}$$Desglosemos la suma que acabamos de ver: $$\begin{align*}
&\bullet \ \text{Cuando } r = 0, \ \text{el término es } \binom{p}{0} a^p b^0 = a^p. \\
&\bullet \ \text{Cuando } r = p, \ \text{el término es } \binom{p}{p} a^0 b^p = b^p. \\
&\bullet \ \text{Para } 0 < r < p, \ \text{los términos son } \binom{p}{r} a^{p-r} b^r.
\end{align*}$$Por la parte $\text{a)}$, sabemos que cada término $\binom{p}{r} a^{p-r} b^r$ con $0 < r < p$ es divisible por $p$, y por lo tanto se anulan en la suma y solo quedan los términos con $r=0$ y $r=p$. Dado que $\binom{p}{0}=1$ y $\binom{p}{p}=1$, se llega a la expresión que buscábamos demostrar.

##### **Teorema 4.3.2 Pequeño Teorema de Fermat**
Sea $p$ primo y $a$ entero, entonces $$a^p \equiv a \pmod{p}$$**Demostración.** Supongamos que $a\geq0$, entonces hagamos inducción en $a$. Si $a = 0$, el resultado es trivial. Supongamos el resultado probado para $k$, es decir $kp ≡ k \pmod{p}$. Entonces $(k+1)^p ≡ k^p+1^p ≡ k+1 \pmod{p}$ . La primera congruencia es debido al lema **4.3.1 b)** y la segunda es válida por hipótesis inductiva. Luego $a^p ≡ a \pmod{p}$ cuando $a > 0.$
Si $a>0$, entonces $−a > 0$ y ya vimos que $(−a)^p ≡ −a \pmod{p}$, es decir que $(−1)^pa^p≡ (−1)a \pmod{p}$. Si $p\neq 2$, entonces $(−1)^p = −1$ y se deduce el resultado. Si $p = 2$, entonces $(−1)^p = 1$, pero como $1 ≡ −1 \pmod{2}$, (restemos $1-(-1)$ para mejor comprensión) obtenemos también $a^p ≡ a \pmod{p}$.

###### **Corolario 4.3.3.** Sea $p$ primo y $a$ entero tal que $p \nmid a$, entonces $a^{(p−1)} ≡ 1 \pmod{p}.$
**Demostración.** Por Fermat $a^p \equiv a \pmod{p}$, es decir $$p | (a^p - a) = a (a^{p-1} - 1)$$Como $p$ no divide a $a$, tenemos que $p |  (a^{p-1} - 1)$, por el teorema **3.4.6**, es decir $a^{p-1} \equiv 1 \pmod{p}$. 
***Este último corolario es también conocido como teorema de Fermat.***
